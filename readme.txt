Данная программа представляет из себя приложение с мини игрой.
После авторизации игроку предложены скрытые числа, после нажатия на которые
число раскрывается и добавляется в общую сумму. После четырех выбранных чисел
игра завершается и игрок может видеть свой результат. После просмотра результата
числа обнуляются и игру можно начать заново. Также имеется программа-спутник
которая препятствует подключению к программе игры с помощью отладчика.
Также разработано хранение пин кода для входа в виде хэш суммы в файле, с которым
происходит сверка в момент ввода пин кода.
